# 🛡️ SafeHer – Women's Safety & Emergency Assistance Platform

## Overview

SafeHer is a Flask-based web application focused on improving women's safety through emergency assistance, crime reporting, safe route planning, transportation support, and safety awareness tools.

The platform helps users travel more safely, report incidents, access emergency services, and stay informed about their surroundings.

---

## ✨ Features

* 🚨 **SOS Emergency Alert** – Send emergency alerts and location details to registered contacts.
* 📝 **Crime Reporting** – Report incidents with supporting evidence.
* 🗺️ **Crime Map** – View reported crime locations on an interactive map.
* 🛣️ **Safe Route Planner** – Find safer routes based on crime data and area safety.
* 🚌 **SHE Shuttle Finder** – Explore available shuttle routes and transportation options.
* 🤖 **Safety Chatbot** – Get guidance on women's safety and legal rights.
* 👤 **User Management** – Registration, login, profile, and emergency contacts.
* 🔒 **Admin Dashboard** – Manage users and review reported incidents.

---

## 🛠️ Tech Stack

**Backend:** Python, Flask, SQLite

**Frontend:** HTML, CSS, JavaScript

**Maps & Location:** Leaflet.js, OpenStreetMap, Geolocation API

**Other:** Email-based SOS alerts, File Uploads

## ⚙️ Installation

```bash
git clone https://github.com/your-username/SafeHer.git
cd SafeHer

python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

pip install -r requirements.txt

python app.py
```

Open:

```text
http://127.0.0.1:5000
```
